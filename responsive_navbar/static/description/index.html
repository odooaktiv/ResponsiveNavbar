Responsive Navbar

Using this module the navbar in website will dynamically hide menus in a dropdown when there is less space instead of moving to the new line. This module also has feature to manage multi-level submenu. The submenus has feature to adjust to left, right or center depending on space available.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

No configurations needed.

Features demonstration

When You install this module It will make the navbar responsive in web view by hiding extra menus if no space is available.

In the first image you can see how odoo default navbar looks when there are more menus then the available space.

In the demo you can see that ODOO does not have proper design solution for multi-level hierarchy of menus.

After installation the + icon appears which contains dropdown of the menues which did not fit in the view.

In the demo you can see that after installation of the module the menus opens properly until the last available sub-menu.
